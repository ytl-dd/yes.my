  1. Apa itu YES Databack x Shopee?
    Dengan setiap pembelian RM50 di Shopee, anda berhak mendapat 5GB data YES secara percuma. Setiap pembelian RM50 yang seterusnya akan memberikan anda data tambahan 5GB lagi secara percuma, dan data percuma yang boleh diterima adalah sehingga 100GB maksimum. Tertakluk kepada terma dan syarat. Misalnya, sekiranya anda berbelanja RM100 pada bulan Januari, anda akan menerima 10 GB data percuma pada bulan Februari.
  2. Apakah had tuntutan baucar YES Databack x Shopee bagi setiap ID Shopee?
    Tidak ada had yang ditetapkan untuk jumlah baucer yang boleh dituntut untuk setiap ID Shopee. Namun begitu, baucar YES Databack x Shopee daripada satu (1) ID Shopee hanya boleh dituntut untuk satu (1) akaun/nombor telefon perkhidmatan YES. Sebaik sahaja anda menuntut YES Databack daripada mana-mana satu ID Shopee bagi akaun/nombor telefon YES, ID Shopee tersebut akan di pasangkan dengan akaun/nombor telefon tersebut. Ini bermakna, bagi tuntutan seterusnya, baucar Shopee yang diterima oleh mana-mana satu ID Shopee itu hanya boleh digunakan untuk akaun/nombor telefon yang sama.
  3. Apakah tempoh sah baucar YES Databack x Shopee?
    Tempoh sah baucar YES Databack x Shopee adalah 4 bulan dari tarikh ia dikeluarkan. Sebaik sahaja diaktifkan, data tersebut perlu digunakan dalam tempoh 30 hari. 
  4. Saya tidak menerima sebarang baucar YES Databack x Shopee. Bagaimanakah cara untuk saya mendapatkannya?

    Untuk mendapatkan baucar YES Databack, pengguna Shopee perlu berbelanja sekurang-kurangnya RM50. Setiap pembelian RM50 yang seterusnya melayakkan anda untuk 5GB lagi data percuma. Anda boleh mendapat sehingga maksimum 100GB data percuma.
     

    Baucar YES Databack x Shopee akan dikredit secara automatik ke akaun Shopee anda. Anda boleh mengambil semua baucar ini daripada Yes Circle melalui aplikasi Shopee.


    Sila rujuk kepada sejarah perbelanjaan bulanan anda (misalnya 1 Jun - 30 Jun). Sekiranya anda berbelanja RM50 ke atas, anda bolehlah mendapatkan baucar YES Databack x Shopee.
     

    Sekiranya anda tidak menjumpainya, tetapi layak untuk mendapat baucar berdasarkan jumlah pembelian yang dilakukan, sila rujuk kepada Perkhidmatan Pelanggan Shopee untuk bantuan selanjutnya.
     

    Baucar YES Databack x Shopee ditawarkan semasa stok masih ada dan tertakluk kepada Terma & Syarat (dipaparkan di Shopee) Promosi YES Databack x Shopee.

  5. Saya ialah pengguna YES. Bagaimanakah cara untuk saya menuntut baucar YES Databack x Shopee?

    Untuk mengakses dan mendapatkan baucar YES Databack x Shopee anda daripada aplikasi Shopee, sila ikuti langkah-langkah berikut:
    1. Lancarkan aplikasi Shopee
    2. Sila masuk Notifications (Pemberitahuan)
    3. Cari maklumat kemas kini Shopee
    4. Tuntutlah baucar YES Databack x Shopee
     

    Seterusnya, muat turun/lancarkan aplikasi MyYES dan ikuti langkah-langkah berikut:

    1. Klik “Have a Voucher? Redeem Here” di laman utama aplikasi My YES
    2. Masukkan nombor siri baucar serta ID Shopee dan klik butang “Verify”
    3. Sebaik sahaja penembusan anda berjaya, data percuma yang diperoleh akan dikredit secara automatik ke dalam akaun YES anda.

  6. Apakah tempoh sah data YES percuma sebaik sahaja ia dituntut melalui aplikasi MyYES?Data percuma yang diperoleh adalah sah untuk 30 hari dari tarikh ia dituntut.
  7. Saya sudahpun mendaftar dengan pelan YES yang menawarkan Data Tanpa Had (Unlimited Data). Apakah manfaat baucar YES Databack x Shopee kepada saya?
    Sekiranya anda sudahpun mempunyai pelan YES Unlimited Data, baucar YES Databack x Shopee tidak boleh dituntut. Namun, anda boleh menuntut baucar ini sebaik sahaja Pelan anda tamat.
  8. Data MyYes Databack tidak tersenarai dalam Akaun YES saya. Apakah yang perlu saya buat?
    Pastikan pelan perkhidmatan YES anda adalah aktif untuk menikmati data percuma.  Sekiranya data percuma ini masih tidak tersenarai dalam Akaun YES anda, sila hubungi YES Care melalui emel  di yescare@yes.my 
  9. Bolehkah baucar YES Databack x Shopee digunakan secara kumulatif?
    Ya, baucar ini boleh digunakan secara kumulatif.
  10. Bolehkah saya berkongsi ataupun memberikan baucar YES Databack x Shopee kepada kawan-kawan atau ahli keluarga saya?
    Baucar YES Databack x Shopee daripada satu (1) ID Shopee hanya boleh dituntut menerusi satu (1) akaun/nombor telefon YES.

    Justeru, sekiranya anda sudahpun menuntut baucar melalui akaun/nombor telefon YES anda sebelum ini, anda tidak akan dapat menuntut lagi melalui akaun/nombor telefon yang lain.

    Sekiranya anda menuntut baucar YES Databack x Shopee buat kali pertama, pastikan anda menggunakan akaun/nombor telefon YES yang terpilih.
